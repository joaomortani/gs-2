# Projeto Global Solutions FIAP 2025

GLOBAL SOLUTION 2025 — O FUTURO DO TRABALHO
Desafio:
Desenvolva uma solução digital composta por um aplicativo mobile híbrido (React Native) integrado a uma plataforma web (React ou Next.js) que explore a temática “O Futuro do Trabalho”. O projeto deve propor informações, conexões ou ferramentas que facilitem a vida de pessoas do mercado de trabalho, considerando os impactos da tecnologia, da automação e das novas formas de trabalho. O grupo deverá criar uma persona representando o público-alvo da solução.
Exemplos: 	profissionais em recolocação, pessoas idosas em processo de atualização digital, trabalhadores autônomos ou freelancers.
Requisitos técnicos obrigatórios:
Aplicativo desenvolvido em React Native (recomenda-se o Expo).
Plataforma web desenvolvida em React ou Next.js, conectada ao app por meio de API ou sincronização de dados.
Integração funcional entre o app e a plataforma web, demonstrando compartilhamento de informações (exemplo: cadastros, consultas, listas, dashboards).
Interatividade implementada em ambas as partes do projeto, com uso de estado e eventos.
Acessibilidade aplicada e justificada tanto no app quanto na plataforma web.
Design responsivo e experiência do usuário coerente entre as duas interfaces.
Organização adequada de pastas, componentes e lógica de integração (front-end e back-end, se houver).
Entrega:
O grupo deverá enviar um único PDF pela plataforma on.fiap.com.br, contendo:
Explicação e justificativa da solução em relação à persona e à temática.
Descrição dos recursos de acessibilidade implementados e como foram testados.
Descrição das principais telas, fluxos e como ocorre a integração entre app e plataforma web.
Links dos repositórios GitHub (mobile e web).
Links de deploy ou execução online, podendo incluir Snack Expo, Vercel ou outro serviço equivalente.
Critérios de avaliação:
Desenvolvimento técnico e qualidade do código (estrutura, componentização, integração, estado e eventos).
Interatividade funcional e relevância para a persona.
Integração entre as plataformas (mobile + web) de forma coerente e funcional.
Acessibilidade aplicada com justificativa.
Aderência à temática “O Futuro do Trabalho”.
Experiência do usuário (clareza, fluxo, responsividade e consistência entre as interfaces).
Criatividade e solução disruptiva.
